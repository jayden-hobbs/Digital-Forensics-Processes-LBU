| Type                    | Attribute Type Identifer | Description                                                                                                      |
| ----------------------- | ------------------------ | ---------------------------------------------------------------------------------------------------------------- |
| `$STANDARD_INFORMATION` | 0x10                     | General information, such as flags; the last accessed, written, and created times; and the owner and security ID |
| `$FILE_NAME`            | 0x30                     | File name, the last accessed, written, and created times                                                         |
| `$OBJECT_ID`            | 0x40                     | A unique identifier for the file or directory                                                                    |
| `$SECURITY_DESCRIPTOR`  | 0x50                     | The access control and security properties                                                                       |
| `$DATA`                 | 0x80                     | File contents                                                                                                    |
